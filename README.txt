🧩 Projet : Matching
Matching signifie « correspondance ». L’idée est simple : un utilisateur effectue une recherche, et l’application lui propose des résultats en lien avec ses critères.

🎯 Objectif
Développer un site web de matching, avec deux fonctionnalités principales :

Créer un contenu qui pourra apparaître dans les recherches

Faire une recherche pour afficher les contenus correspondants

🧠 Thème 100 % libre
Le thème de votre application est entièrement libre. Cela peut être, par exemple :

Un site de rencontres

Une plateforme de vente d’occasion

Un outil de recherche d’emploi

Un système de mise en relation autour d’une passion ou d’un besoin

Etc.

L’essentiel est d’imaginer un cas d’usage où le matching est pertinent, puis de concevoir les critères de création et de recherche adaptés à ce contexte.

Exemple d'application :
Rencontres pour les amateurs de promenade
Critères possibles :

Lieu de départ

Distance souhaitée

Type de promenade (à pied, à vélo…)

Niveau de sportivité

🔎 Fonctionnalité de recherche
Votre système de recherche devra intégrer au moins 3 critères différents, tous facultatifs.
L’utilisateur doit pouvoir effectuer une recherche :

avec 1 seul critère

avec 2 critères

avec 3 critères

Le système doit renvoyer des résultats pertinents dans tous les cas.
